Essential Commands
  • ! command Instant bash execution (no token cost)
  • # prefix Save info to Claude's memory
  • & prefix Send long tasks to Claude on the Web
  • /init Auto-generate CLAUDE.md with project context
  • /export Export session to Markdown
⌨
Shortcuts & Navigation
  • Esc Esc Rewind/undo last turn
  • /vim Enable Vim-style editing
  • Ctrl+S Stash draft prompt (auto-restores)
  • Ctrl+R Reverse search prompt history
  • Tab / Enter Accept predictive follow-ups
↺
Workflow & Sessions
  • --continue Resume last session
  • --resume Pick from past sessions
  • /rename Name session for easy recall
  • claude -p "..." Headless mode for scripts
  • "plan" Analyze & plan before executing
★
Thinking & Context
  • "think" Light reasoning mode
  • "hard" / "ultrathink" Deep reasoning (up to 32k)
  • /context Visualize token usage & space
  • Prune MCP tools Remove unused tools to save context
  • /stats View your activity graph
💡
Customization
  • /statusline Configure custom status bar
  • Hooks Trigger actions on events
  • Theme Customize Claude's appearance
  • Keybindings Remap keys for your workflow
  • Notifications Alerts when Claude needs attention
🔒
Tools & Extensions
  • Browser Navigate, click, read, fill forms
  • SKILL.md Teach Claude new capabilities
  • Plugins Bundle commands, agents, skills
  • MCP Servers Enable/disable context protocols
  • LSP Instant diagnostics & code nav
